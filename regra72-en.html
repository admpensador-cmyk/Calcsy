Calculadora da Rule of 72

How does this calculator work?

A Rule of 72 é um atalho mental para estimar em quanto tempo um valor tende a dobrar com compound interest, usando uma taxa anual aproximada.

Fórmula (aproximação)

Tempo (anos) ≈ 72 / taxa (%)

Quando usar

  • Para comparações rápidas entre taxas (ex.: 8% vs 12%).
  • Para ter noção de prazos (dobrar em 6, 9, 12 anos...).

Importante

É uma estimativa. Para cálculos precisos, use a fórmula exata: n = ln(2) / ln(1 + i).
